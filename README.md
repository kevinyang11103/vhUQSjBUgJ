## 二、技术栈

### 基础技术

- **Python**: 后端逻辑编写，负责处理用户请求、与数据库交互、生成动态内容。
- **SQLAlchemy**: 后端数据存储工具，提供对象关系映射（ORM）功能，简化数据库操作。
- **Django 框架**: 利用其路由管理、模板引擎、表单验证等功能，搭建网站后端。

### 安全性

- 使用 Django 表单验证，实现前后端数据校验，防止恶意输入和数据错误。
